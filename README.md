# AI Business Growth OS

Welcome to **AI Business Growth OS** — a full-stack, AI-powered orchestration system designed to act as an autonomous growth team for startups and businesses. 

## 🚀 Overview

The system ingests your raw business data (CSV uploads, CRM data, marketing ad spend) and runs it through a robust **5-Agent Orchestration Pipeline** using FastAPI and advanced Large Language Models. These specialized agents work together to generate a comprehensive, actionable growth strategy.

### The 5 AI Agents:
1. **Analytics Agent**: Processes your raw data, builds statistical profiles, and identifies churn risks and revenue opportunities.
2. **Strategy Agent**: Reads the Analytics output and formulates high-level pivot and scaling strategies.
3. **Marketing Agent**: Takes the strategic goals and optimizes ad spend (ROAS) across Meta and Google platforms.
4. **Sales Agent**: Designs sales funnels, lead scoring mechanisms, and closing tactics.
5. **Customer Success Agent**: Focuses on retention, support workflows, and mitigating identified churn risks.

## ✨ Key Features

- **Dynamic Data-Driven Dashboard**: Beautiful, premium React frontend that automatically updates its KPIs, charts, and risk alerts based on the real-time insights generated by the backend agents.
- **AI Copilot**: A conversational interface that allows you to directly ask the AI questions about your data (e.g., "Why is my Customer Health Score dropping?"). The Copilot retains session memory across navigation.
- **Long-Term LLM Memory**: Raw CSV uploads are permanently stored in MongoDB. The Orchestrator injects statistical summaries and past predictions into the AI context, allowing the LLM to compare its past predictions against your new actual outcomes and learn over time.
- **Marketing Performance Tracker**: Input real-time ad spend and revenue data, which is instantly piped into the AI orchestration engine to adjust strategies dynamically based on ROAS.

**Backend:**

- JWT (Authentication)
- Nvidia AI Endpoints (Llama-3/Mistral LLM Integration)

## ⚙️ Local Setup

### 1. Clone the repository
```bash
git clone https://github.com/preethamkr1/ai-business-os.git
cd ai-business-os
```

### 2. Backend Setup
```bash
cd backend
pip install -r requirements.txt
```
Make sure you have MongoDB running locally (or update `mongo_service.py` with your Mongo URI). 
Set your Nvidia API Key in the `.env` file or environment variables.
```bash
uvicorn app.main:app --reload
```

### 3. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

Visit `http://localhost:5173` to view the application.
